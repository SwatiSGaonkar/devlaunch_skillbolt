# 🚀 DevLaunch

DevLaunch is a full-stack MERN Job Portal that connects students with recruiters. Users can register, log in, browse jobs, save jobs, apply for positions, and recruiters can manage applications from a dedicated dashboard.

---

## ✨ Features

### 👨‍🎓 Student
- Register & Login
- Browse Jobs
- Search Jobs
- Save Jobs
- View Job Details
- Apply for Jobs
- Profile Dashboard

### 👨‍💼 Recruiter
- View Applications
- Accept Applications
- Reject Applications
- Track Application Status
- Recruiter Dashboard

---

## 🛠️ Tech Stack

### Frontend
- React

- React Router
- Axios
- Framer Motion

### Backend
- Node.js
- Express.js
- MongoDB Atlas
